OpenAI API:

1. platform.openai.com saytida ro'yxatdan o'ting

2. API Keys bo'limiga o'ting

3. "Create new secret key" tugmasini bosing va kalitni saqlang

4. Kalitni yuqoridagi maydoncha kiriting

Claude API (Anthropic):

1. console.anthropic.com saytida ro'yxatdan o'ting

2. API Keys bo'limiga o'ting

3. "Create Key" tugmasini bosing va kalitni saqlang

4. Kalitni yuqoridagi maydoncha kiriting

Eslatma: API kalitlaringizni hech qachon boshqalar bilan ulashmang. Ular faqat brauzeringizda saqlanadi va serverga yuborilmaydi.
